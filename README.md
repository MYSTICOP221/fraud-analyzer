# Fraud Transaction Analyzer

A Python project that analyzes financial transactions, detects suspicious activity, and visualizes spending patterns. Designed to simulate real banking transaction analytics, relevant for IT and risk management roles.

---

## Features

- Reads transaction data from CSV files  
- Calculates total and category-wise spending  
- Detects suspicious transactions using:
  - **Dynamic thresholds** (mean + 2× standard deviation)
  - **Rapid transactions** (multiple transactions in same category on same day)
- Visualizes spending by category with bar charts  
- Optional export of suspicious transactions to CSV for reporting
